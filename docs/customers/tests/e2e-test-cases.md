# Casos de Teste - Clientes (Customers)

Este documento define os cenários de teste E2E para validação das regras de negócio associadas ao cadastro, edição e autorização logística de clientes.

---

Clicar em "Salvar Cliente". | - Modal fecha.<br>- Toast: "Cliente atualizado com sucesso!".<br>- Nome atualizado na tabela.<br>- Badge "Bi-truck" aparece junto aos anteriores. |
